Bitcoin Ghost is a complete infrastructure layer for Bitcoin. Decentralized mining, node incentives, instant payments, and privacy tools — all without changing the protocol.

What is Bitcoin Ghost?

Infrastructure that rewards participation

Ghost makes it profitable to run Bitcoin infrastructure. The node reward pool is funded from mining pool and L2 transaction fees, and rewards are shared based on the services each node provides — verified by peer challenges across the network.

⛓️
Decentralized Mining
Every node runs its own pool. Miners connect via standard Stratum V1/V2. No central pool operator controls block construction or reward distribution.
⚡
Instant Payments
Ghost Pay processes transactions in ~10 seconds using shielded notes and zero-knowledge proofs. No channels, no liquidity management. Settles to Bitcoin L1.
🔒
Privacy by Default
Silent Payments, CoinJoin mixing, relay timing resistance, and blocks-only mode. Privacy tools at every layer, from transaction relay to payment settlement.
Features

Everything built and running on signet

Mining & Pool
Ghost Pool
Decentralized mining pool built into every node. Each node constructs its own block templates.
Stratum V1 & V2
Standard mining protocol support. Any ASIC can connect immediately.
BFT Payout Consensus
67% supermajority agreement on reward distribution before block inclusion.
Template Distribution
Node controls complete coinbase construction including miner payouts and node rewards.
Node Incentives (5-4-3-2-1 Shares)
Archive Mode (+5)
Full block history with witness data. Verified by random block retrieval challenges.
Ghost Pay (+4)
Run the L2 instant payment service. Verified by L2 block lookup challenges.
Public Mining (+3)
Accept external miners via Stratum. Verified by port accessibility probes.
Bitcoin Pure (+2)
Clean block policy filtering spam transactions. Verified by classification challenges.
Elder Status (+1)
First 101 registered nodes. Permanent bonus for early participants.
Privacy
Ghost Mode
Blocks-only relay preventing origin address leakage from mempool analysis.
Ghost Shroud
Random relay delays defeating timing correlation attacks.
Ghost Wraith
CoinJoin mixing protocol with blind signing for transaction unlinkability.
Ghost Haze
Stripped block storage removing witness data from historical blocks.
Silent Payments (BIP-352)
Stealth addresses. One static address, unique one-time outputs, no address reuse.
Ghost Tor
Embedded Tor subprocess routing all P2P traffic through the Tor network.
L2 Payments
Ghost Pay
Instant payment layer with ~10 second settlement using ZK proofs.
Ghost Locks
Timelocked P2TR outputs with key rotation and L1 settlement.
Ghost ID
Silent Payment identity for receiving L2 payments without being online.
Policy & Classification
BUDS (T0-T3)
Bitcoin Unified Data Standard classifying transactions into 4 tiers for policy filtering.
Template & Mempool Profiles
Pre-built policies: Bitcoin Pure, Permissive, Full Open. Custom profiles supported.
Ghost Reaper
Spam transaction filtering engine with configurable rules.
Wallets & Tools
Ghost-Tap Desktop
Full node or light wallet mode with merchant terminal and L2 management.
GSP Light Wallet
WebSocket protocol for light wallets without running a full node.
Node Dashboard
Web-based operator monitoring with real-time metrics and WebSocket updates.
Node TUI
Terminal dashboard for headless server management over SSH.
