✓ Verify Your Download

Always verify the signature of downloaded files to ensure authenticity and integrity.

# Import the Ghost signing key

Configuration Example

Sample pool.toml for a typical node operator (v0.1.0).

# pool.toml - Ghost Node Configuration v0.1.0
# https://github.com/bitcoin-ghost/ghost-node

# === Network ===
mainnet=1
testnet=0

# === Ghost Mode (private relay) ===
ghostmode=0

# === Node Rewards (5-4-3-2-1 system) ===
# Archive Mode: +5 shares
archive=1

# Public Mining: +3 shares
public_mining=1
stratum_v1_port=3333
stratum_v2_port=3334

# === Pruning (VW fixed at 576 blocks) ===
prune_ow=2016
prune_profile=default

# === Mempool & Template Profiles ===
mempool_profile=structured
template_profile=max_fee

# === Payout ===
payout_address=bc1q...your_address_here
min_payout=0.001
