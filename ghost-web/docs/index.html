Welcome.

Bitcoin Ghost is built on a modified Bitcoin Core fork — the consensus rules are unchanged, the mining stack and L2 are added on top. We inherit Bitcoin's security model, then layer a decentralised mining pool and a shielded L2 payment network around it. The docs on the left cover everything from first principles to the RPC interface.

New to Ghost? Start with Ghost Core for the node-level view, then Ghost Pool for how the mesh earns rewards, and Economics deep-dive for the full reward-split picture.

Running a node? Jump to Node dashboard and Elder system.
