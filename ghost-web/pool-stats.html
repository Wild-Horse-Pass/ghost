Stats API

All stats are available via our public API. No authentication required.

GET https://pool.bitcoinghost.org/api/v1/network/pool Pool overview stats
GET https://pool.bitcoinghost.org/api/v1/blocks?limit=50 Recent blocks
GET https://pool.bitcoinghost.org/api/v1/miners/{address} Miner stats by address
GET https://pool.bitcoinghost.org/api/v1/nodes Node health status
