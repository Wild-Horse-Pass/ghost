# ghost-pool

Mining pool functionality for the Ghost Network.

## Overview

This crate implements Stratum V2 compatible mining pool functionality, including job distribution, share validation, and miner management.

## Features

- **Stratum V2**: Modern mining protocol with improved efficiency
- **Stratum V1**: Backwards compatibility for legacy miners
- **Variable Difficulty**: Automatic difficulty adjustment per miner
- **Job Management**: Efficient block template distribution
- **Share Validation**: Fast share verification with proper target handling

## Components

| Component | Description |
|-----------|-------------|
| Job Manager | Creates and distributes mining jobs |
| Share Validator | Validates submitted shares against target |
| Miner Manager | Tracks miner connections and statistics |
| Difficulty Adjuster | Tunes difficulty for consistent share rate |

## Usage

```rust
use ghost_pool::{PoolCoordinator, PoolConfig};

let config = PoolConfig::default();
let pool = PoolCoordinator::new(config);

// Start the pool
pool.start().await?;

// Get pool statistics
let stats = pool.stats();
println!("Connected miners: {}", stats.connected_miners);
```
