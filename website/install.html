Install Bitcoin Ghost

Quick Install (Recommended)

🚀

One-Line Install

Install Bitcoin Ghost with a single command:

curl -sSL https://bitcoinghost.org/install.sh | bash

This will download and install the latest version of ghost-pool.

Docker Installation

🐳

Docker Compose

Run with Docker for easy deployment:

1 Download docker-compose.yml 
curl -O https://bitcoinghost.org/downloads/docker-compose.yml
2 Create .env file 
cat > .env << 'EOF'
BITCOIN_RPC_USER=ghost
BITCOIN_RPC_PASSWORD=ghostpass
BITCOIN_NETWORK=signet
TREASURY_ADDRESS=tb1qyouraddress
LOG_LEVEL=info
EOF
3 Start the pool 
docker-compose up -d
4 Check logs 
docker-compose logs -f ghost-pool

Manual Installation

📦

Download Binary

Linux x86_64 ghost-pool-linux-x86_64.tar.gz SHA256
Linux ARM64 ghost-pool-linux-arm64.tar.gz SHA256

Installation Steps

1 Download and extract 
wget https://bitcoinghost.org/downloads/ghost-pool-linux-x86_64.tar.gz
tar xzf ghost-pool-linux-x86_64.tar.gz
sudo mv ghost-pool /opt/ghost/bin/
2 Create configuration 
sudo mkdir -p /etc/ghost
sudo nano /etc/ghost/pool.toml
3 Create systemd service 
sudo tee /etc/systemd/system/ghost-pool.service << 'EOF'
[Unit]
Description=Ghost Mining Pool (SV2)
After=network.target bitcoind.service

[Service]
Type=simple
User=ghost
ExecStart=/opt/ghost/bin/ghost-pool --config /etc/ghost/pool.toml
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
EOF

sudo systemctl daemon-reload
sudo systemctl enable ghost-pool
sudo systemctl start ghost-pool

Configuration Example

# ghost.toml - Bitcoin Ghost Pool Configuration

[bitcoin]
rpc_host = "127.0.0.1"
rpc_port = 38332          # signet RPC port
rpc_user = "ghost"
rpc_password = "ghostpass"
network = "signet"
zmq_hashblock = "tcp://127.0.0.1:28332"

[network]
public_address = "your.server.ip"
sv2_port = 34255          # Stratum V2
sv1_port = 3333           # Stratum V1 (via translator)
http_port = 8080          # API/Dashboard

[policy]
profile = "permissive"    # or "bitcoin_pure", "full_open"

[storage]
data_dir = "~/.ghost"
archive_mode = false      # Set true for full history

[pool]
treasury_address = "tb1qyouraddress"
fee_percent = 1.0
min_payout_sats = 100000

Requirements

  • ✓ Bitcoin Core (or ghost-core) with signet
  • ✓ Linux x86_64 or ARM64
  • ✓ 4GB RAM minimum
  • ✓ 50GB disk (signet) / 500GB+ (mainnet)
  • ✓ Ports 3333, 8080 accessible
